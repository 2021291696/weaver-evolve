# 常见错误模式速查

每种错误类型的高频根因和解法方向。

## 语法错误

| 模式 | 根因方向 | 预防方向 |
|------|----------|----------|
| `SyntaxError: invalid syntax` | 改 A 没同步 B | 改代码后跑 linter |
| `NameError: name 'x' is not defined` | 变量拼写/作用域 | IDE 自动补全 |
| 类型不匹配 | 函数签名改了但调用方没改 | 类型检查 CI 步骤 |

## 配置错误

| 模式 | 根因方向 | 预防方向 |
|------|----------|----------|
| `ModuleNotFoundError` | 依赖未安装/虚拟环境 | pip install 前检查 |
| 端口占用 | 上次进程没杀 | 启动前 kill 旧进程 |
| 环境变量未设 | .env 文件缺失 | 模板 .env.example |

## 逻辑错误

| 模式 | 根因方向 | 预防方向 |
|------|----------|----------|
| 死循环 | 退出条件永远不满足 | while 设最大迭代 |
| 状态机卡死 | 异步操作失败后状态未复位 | finally 块复位 |
| null/undefined | 链式调用中间断掉 | optional chaining |

## 环境错误

| 模式 | 根因方向 | 预防方向 |
|------|----------|----------|
| 编码乱码 | Windows GBK vs UTF-8 | ASCII 优先 |
| 路径不存在 | 硬编码路径/中文路径 | 路径用变量 |
| 杀软拦截 | Defender 扫描大 exe | 重命名绕过 |

## 第三方错误

| 模式 | 根因方向 | 预防方向 |
|------|----------|----------|
| API 变更 | 库升级了没看 changelog | 锁定版本号 |
| 库内部异常 | 库的边界条件 bug | try/except 包裹 |
| 版本冲突 | 两个库依赖不同版本的同一个包 | pip check |
